# GitHub Pages (база)

Это минимальный статический сайт для публикации на GitHub Pages.

## Как опубликовать

1) Создайте репозиторий на GitHub (публичный).
2) Запушьте этот проект в репозиторий.
3) Включите Pages:
   - Settings → Pages
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**

Через минуту страница будет доступна по адресу:

- `https://<ваш-логин>.github.io/<имя-репозитория>/` (для project pages)
- или `https://<ваш-логин>.github.io/` (если репозиторий назван `<ваш-логин>.github.io`)
